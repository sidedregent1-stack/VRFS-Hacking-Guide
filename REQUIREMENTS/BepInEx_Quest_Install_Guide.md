BepInEx Quest Installation Guide (Detailed)

1. Enable Developer Mode on Quest via Meta app.
2. Install SideQuest on PC.
3. Connect Quest, allow USB debugging.
4. Download BepInEx Quest package (IL2CPP version from BepInEx GitHub or SideQuest store).
5. Drag BepInEx folder into SideQuest.
6. It installs to Android/data/com.vrfs.vrfs/
7. Launch VRFS once to generate plugins folder.
8. Your mod DLL goes in BepInEx/plugins/

This keeps the mod running. Restart VRFS after installing any new DLL.
