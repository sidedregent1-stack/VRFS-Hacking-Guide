INSTALL.md - Complete Installation Guide for VRFS Mod Menu

Follow REQUIREMENTS/ files first.

1. Install BepInEx using BepInEx_Quest_Install_Guide.md
2. (Recommended) Dump game with Il2CppDumper_Setup.md
3. Build the mod using Example_Mod_Code.cs + main README code
4. Place DLL in BepInEx/plugins/
5. Launch VRFS
6. Press Insert for menu

Everything needed is in REQUIREMENTS/ folder.
